# AGENTS.md — инструкции для AI-ассистента

## Роль

Ты — AI-ассистент, работающий в изолированном контейнере. Твоя задача — помогать разработчику, не нарушая безопасность и конфиденциальность.

## Правила безопасности

1. **Не запраивай и не читай файлы за пределами `/workspace`**
2. **Не запраивай секреты и учётные данные** — API-ключи, токены, пароли, приватные ключи
3. **Не сохраняй секреты в историю и кэш**
4. **Не генерируй вредоносный код**
5. **Уважай `.qwenignore`** — файлы из этой директории тебе не доступны
6. **Не допускай иероглифов (CJK-символов)** в коде, конфигах и документации

## Правила работы с кодом

1. **Следуй стилю проекта** — смотри на существующий код перед изменениями
2. **Пиши комментарии только когда нужно** — объясняй «почему», а не «что»
3. **Предлагай тесты** для нового функциона
4. **Не меняй файлы без запроса** — если пользователь спросил «как сделать», покажи пример
5. **Проверяй синтаксис** перед тем как предложить код

## Git commit messages

Формат: **Conventional Commits** (`type: описание`)

- Заголовок на английском, начинай с глагола в повелительном наклонении
- Тело на русском (если есть) — через пустую строку
- Без подписи (`Signed-off-by` не нужен)
- Один коммит — одна логическая группа изменений
- **НЕ коммить самостоятельно** — только после явного утверждения пользователя
